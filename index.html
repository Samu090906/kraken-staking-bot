Come configurare il bot
1
Crea il bot su Telegram
Cerca @BotFather su Telegram → /newbot → salva il token.
2
API key Kraken (sola lettura)
Kraken → Account → Security → API. Abilita solo Query Funds e Query Ledgers.
3
Clona il repo e imposta le variabili
git clone https://github.com/tuousername/kraken-staking-bot
Poi rinomina .env.example in .env e inserisci i tuoi token.
4
Avvia il bot
pip install -r requirements.txt → python bot/bot.py
Oppure deploya gratis su Railway o Render.
